multiDownloader

download stuff quickly off VIT's course page

by Namit Juneja
please report issues on github

usage

open the course page
select all the files you want to download
click multiDownload button or press Shift+M
check for VIT Course Material folder in your downloads folder.

What's new

• Keyboard shortcuts for select all (Shift+A) and multiDownload (Shift+M).
• Ability to add index numbers to files for easy sorting.
• Ability to segregate files into folders based on CAT1/CAT2/TEE.
• One click select all / all CAT1/CAT2/TEE links.
• UI improvements.
• Bug fixes.
